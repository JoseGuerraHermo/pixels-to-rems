## Run locally for updating or improving
* install dependencies:
```
$ npm install
```

* Run the react code localy in localhost:3000 :
```
$ npm start
```

* Serve locally compiled by webpack:
```
$ npx webpack serve
```

* Build for production:
```
$ npm run build
```
